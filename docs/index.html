oxi is a native, local-first coding agent that talks to whatever model you choose — a local GGUF, an Ollama or LM Studio server, a box you SSH into, a hosted API, or an agent CLI over ACP — without a browser engine and without your code, keys, or history leaving your machine.

Why oxi

The opposite bet from cloud coding tools

The polished commercial tools assume you'll use their cloud and their subscription. oxi assumes the opposite: bring your own model, keep everything local, own your data.

🧠

Any model, no lock-in

Hosted APIs, local servers, oxi-managed GGUF models, or agent CLIs over ACP — switch providers from one control.

🔒

Local-first by design

Settings, sessions, tool execution, SSH credentials, OAuth tokens, and models all stay on your machine. No account needed to use your own models.

⚡

Native, not Electron

Rust + egui, a single binary, no bundled browser engine — and a low idle memory footprint.

🧩

Local & self-hosted models

Run GGUF models oxi downloads for you via llama-server, connect to LM Studio or Ollama, or tunnel to a GPU box over SSH.

🗂️

File explorer + code editor

A multi-tab editor with syntax highlighting, minimap, find/replace, external-change detection, and Git changes opened inline — not just a chat window.

🛠️

Workspace-aware tools

Read, write, edit, grep, find, codebase search, Git inspection, verification commands, MCP servers, and web search/fetch.

🌿

Git, terminal & diffs built in

A source-control panel with stage/commit/push, AI commit messages, inline diffs, and a workspace-rooted PTY terminal.

🎙️

Local voice dictation

Optional microphone dictation using local Whisper models — loaded lazily, unloaded when idle. Nothing leaves the machine.

You fully control it

Your machine. Your models. Your prompt.

oxi ships with a sensible coding-agent prompt — but it's just an editable template. Rewrite it and the agent becomes whatever you need.

✍️

Editable system prompt

One editable template with a {tools_list} placeholder. Keep the coding defaults, or replace them entirely. @-mention files and folders into a message.

🎭

Coding agent, or anything

Point it at a repo as a coding assistant, or repurpose it into a reviewer, researcher, or writing partner.

🚫

No vendor lock-in

Nothing hard-coded by a cloud provider. Your data, keys, and instructions never leave your machine.

Install

Up and running in seconds

Homebrew for macOS (Apple Silicon) and Linux (x86_64), or grab a precompiled archive.

🍺 Homebrew
# macOS (Apple Silicon) & Linux (x86_64)
brew tap maziluiosif/tap
brew install oxi
🦀 From source
# requires the Rust toolchain
cargo run --release

Prefer a manual download? Precompiled archives for macOS, Linux, and Windows are attached to every GitHub release.
